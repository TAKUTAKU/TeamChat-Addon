# TeamChat Addon
チーム対戦チャット勢用のAddon

# 使い方
#### ベータAPIを有効にしてください<br>
「team〇〇」というTagのみ動作<br>
「!p 〇〇」で会話が可能<br>

# ダウンロード

PC　右のReleaseから<br>

スマホ　下or上のReleaseから<br>

下の〇〇.mcaddon(.mcpack)っていう青い奴をクリックすると、 ダウンロードできます！<br>
